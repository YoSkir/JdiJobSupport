# JdiReportMaker \n
 (未完成)為工作上報表所做的程式，目前只適用於公司的excel。

1.可以選取電腦中的excel檔
2. 畫面輸入我要輸入的東西，把輸入填入指定的格
3.最後輸出exel
第二步想完成的功能:

3.填日今天的日期到指定的格
4.這個程式可已先設定好員工姓名，之後員工姓名那格也都會自動填入
第三部:
1.有三個分頁分別對應到我們公司三種excel(日報表 周報表 工時表)
